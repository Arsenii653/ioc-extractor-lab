### Что такое Git
Git — это распределенная система контроля версий. 
Она позволяет отслеживать изменения в коде и файлах проекта,а также работать над проектом из разных мест разным людям. 
Он хранит историю изменений в виде коммитов, что дает возможность увидеть, кто, когда и зачем внес правку. 
Git поддерживает работу с ветками, благодоря чему работу над проектом можно распределить и безопасно изменять главную функцию.

### Инцидент с файлом secrets.txt
инцидент: В репозиторий был случайно добавлен файл secrets.txt, содержащий конфиденциальную информацию.

Поиск: 
- git log --oneline -- secrets.txt — для просмотра истории изменений файла
- git show HEAD:secrets.txt — для просмотра содержимого секрета в коммите

исправление:
1. Удаление файла из индекса Git:
    - git rm --cached secrets.txt
2. Добавление файла в .gitignore:
    - echo "secrets.txt" >> .gitignore
3. Создание нового коммита с изменениями:   
    - git add .gitignore
    - git commit -m "Remove secrets.txt and update .gitignore"
